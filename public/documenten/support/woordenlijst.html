Verklarende woordenlijst

AAA-festivals

Met AAA-festival worden festivals bedoeld waarbij het bezoekersaantal jaarlijks boven de 45.000 bezoekers ligt. Deze festivals behoren tot de top festivals van Nederland.

Analytics

Analytics wordt beschreven als data verkregen door metingen. Deze metingen kunnen zowel online (google analytics*) als offline (turven) gedaan worden.

API

Een API is een stukje software dat ervoor zorgt dat systemen over het web tussen elkaar kunnen communiceren.

Branding

De visuele (en niet-visuele) identiteit die het merk vertegenwoordigd. Denk hierbij o.a. aan logos en marketing materiaal maar ook manier van communicatie.

Frameworks

Een framework is een tool of plan wat gebruikt kan worden bij het beantwoorden van een probleem. Deze tool is vaak onderzocht en wordt gezien als doordacht. Door het gebruik van deze tool resulteert vaak een valide oplossing voor een probleem.

Identiteit

Met identiteit wordt het beeld bedoeld dat personeel over het eigen bedrijf hebben. De identiteit omvat elementen als de visie, missie, kernwaarden, ambities, cultuur en de persoonlijkheid van je organisatie.

Imago

Het beeld dat het algemene publiek over WiSH Outdoor heeft. Dit is het tegenovergestelde van de identiteit.

Itereren

Het herhaaldelijk verbeteren van een product aan de hand van nieuwe testresultaten of onderzoek.

Javascript

Javascript is een programmeer taal waarmee je onder andere complexe taken en animaties kan toevoegen aan je website.

Three.js is een Javascript 3D library waarmee het makkelijker wordt gemaakt om 3D content op een webpagina te krijgen.
